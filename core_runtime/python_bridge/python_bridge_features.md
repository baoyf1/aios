- **引用计数**: 正确管理Python对象的引用计数

## 5. 核心API

### 5.1 初始化和关闭
- **初始化**: `initialize_python(config)`
- **关闭**: `shutdown_python()`
- **状态检查**: `get_python_status()`

### 5.2 模块管理
- **加载模块**: `load_module(name)`
- **卸载模块**: `unload_module(name)`
- **列出模块**: `list_modules()`
- **模块信息**: `get_module_info(name)`

### 5.3 函数调用
- **调用函数**: `call_function(module, function, args)`
- **调用方法**: `call_method(object, method, args)`
- **函数信息**: `get_function_info(module, function)`

### 5.4 对象管理
- **创建对象**: `create_object(type, args)`
- **获取属性**: `get_attribute(object, name)`
- **设置属性**: `set_attribute(object, name, value)`
- **删除对象**: `delete_object(object)`

### 5.5 依赖管理
- **安装依赖**: `install_dependency(package, version)`
- **卸载依赖**: `uninstall_dependency(package)`
- **列出依赖**: `list_dependencies()`
- **检查依赖**: `check_dependency(package, version)`
