## 项目目标

构建企业级 AIOS（AI Agent Operating System），实现：
- 自然语言驱动的意图执行
- 跨应用自动化
- 多Agent协作
- 本地LLM部署（Ollama/国内厂商）

## 技术栈

| 层级 | 技术 |
|------|------|
| 系统层 | C/C++ (多线程/进程/IPC) |
| Agent开发 | Python + LangChain + LangGraph |
| LLM | Ollama / 国内LLM / OpenAI兼容 |
| OS集成 | Win32 API / Linux D-Bus |
