## 技术实现

### 核心技术
- C++17 实现核心存储引擎
- Python 3.10+ 提供高级接口
- 向量存储：集成现有框架（FAISS、Milvus等）
- 本地向量模型：支持本地运行的向量模型
- 远程向量API：支持远程向量模型API
- 记忆检索：基于余弦相似度，支持用户问题驱动检索
- 记忆管理：LRU缓存策略，支持配置文件管理
- 记忆隔离：为不同Agent创建独立记忆空间
- 计数引用系统：基于引用计数总结用户喜好
- 记忆增强：基于LLM的记忆重组和用户喜好总结
- 提示词注入：将用户喜好注入提示词

### 依赖项
- FAISS (Facebook AI Similarity Search)
- NumPy
- PyTorch (可选，用于高级向量操作)
- YAML (用于配置文件管理)
- Ollama (可选，用于本地LLM集成)
