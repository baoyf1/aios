## 技术实现

### 核心技术
- C++17 实现底层性能优化
- Python 3.10+ 提供高级接口
- Ollama 作为本地LLM后端
- OpenAI API 作为远程LLM后端
- FastAPI 提供流式响应支持

### 依赖项
- ollama-python (本地LLM集成)
- openai (远程API集成)
- pydantic (参数验证)
- aiohttp (异步网络请求)
- PyYAML (配置管理)
